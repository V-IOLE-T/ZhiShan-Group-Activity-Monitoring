# 至善管理者使用说明（运营版）

更新时间：2026-02-14  
适用对象：社群管理者、内容运营负责人

这份手册只讲管理动作，不讲技术配置。

## 1. 管理者要做的三件事

1. 做好加精（Pin）：把高价值内容及时加精。  
2. 看懂每日加精汇总和活跃度：识别核心贡献星友和主题趋势。  
3. 用好归档：把群内内容沉淀为可复用资料。

## 2. 加精（Pin）怎么做

## 2.1 什么内容建议加精

优先加精这几类：
- 可复用的方法论、模板、清单
- 高质量答疑（问题清晰 + 回答可执行）
- 有结果的实战复盘（过程 + 结论 + 经验）
- 对群目标有明显帮助的提醒/公告

不建议加精：
- 纯情绪表达
- 无上下文的短句
- 已重复出现的同类内容

## 2.2 建议的加精节奏

- 日常：看到优质内容，当天完成加精。
- 每周：固定一次“补加精”，避免遗漏。
- 每月：回看高频主题，为下月内容运营做准备。

## 3. 如何看昨日加精汇总（每天 09:00）

汇总核心看三点：

1. 昨日新增加精数：看内容质量趋势。  
2. 被加精星友分布：看是否过度集中在少数人。  
3. 被加精主题方向：看是否符合当期运营重点。

管理动作建议：
- 新增加精偏少：主动发起主题征集或示范贴。
- 贡献过于集中：点名鼓励腰部星友发言。
- 主题跑偏：次日给出明确话题引导。

## 4. 如何看活跃度（宏观）

活跃度不是只看“谁发得多”，而是看“谁贡献有效”。

重点看：
- 核心星友：持续有输出且有互动反馈的人
- 成长星友：近期明显上升的人
- 沉默星友：长期无参与的人

管理动作建议：
- 对核心星友：公开认可 + 邀请带主题
- 对成长星友：给予更具体的任务和舞台
- 对沉默星友：定向点名提问，降低参与门槛

## 5. 归档怎么服务管理

标签归档不是“存档”，而是“下次可复用”。

建议每周做一次：
- 检查本周重点标签下的内容完整度
- 补充目录或导航，方便星友查找
- 将高频问题沉淀成固定入口（问答合集/攻略合集）

## 6. 管理节奏（可直接照着做）

每日 5 分钟：
- 浏览当天消息，完成应加精内容
- 记录 1-2 条可复用内容

每周 20 分钟：
- 汇总看每日加精趋势，确定下周运营动作
- 发布一次“本周精华回顾”

每月 30 分钟：
- 回看高价值主题和贡献星友
- 设定下月内容方向和激励点

## 7. 常见管理问题

- 觉得没人可加精：通常是标准不清，先明确“什么算高价值”。
- 汇总有数据但讨论不热：说明有沉淀无扩散，补“精华回顾+引导讨论”。
- 活跃度高但质量一般：提高加精门槛，用“示范内容”拉标准。
