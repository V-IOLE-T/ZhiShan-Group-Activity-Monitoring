# 至善星友使用说明（发布版）

更新时间：2026-03-06
适用对象：至善全体星友

这套自动化你可以当成“群内小助手”，核心就 4 件事：
1）记活跃度；2）按标签归档；3）记录加精（Pin）；4）单聊帮你提炼文档。

## 一句话上手

在群里发内容时带 `#标签`，在单聊里给机器人发文档链接。

`[截图建议-01：群里带标签发言示例]`

## 你会用到的功能

## 1. 标签归档

在群消息里带标签，系统会自动把内容归到对应文档。

当前支持：

- `#问答`
- `#打卡`
- `#雅思/英语学习`
- `#AI实用分享`
- `#写作运营`
- `#沟通场景/技巧`
- `#个人思考`
- `#攻略分享`

补充：

- 不带标签不会归档。
- 一条消息带多个标签时，系统会按规则选一个标签归档。

`[截图建议-02：文档里的归档效果示例]`

## 2. 活跃度统计

系统按月统计这些行为：

- 发言、字数、被回复、被@、发起话题
- 点赞、被点赞
- 被加精（Pin）

在云端中心页面可以实时查看到个人的活跃度值和排行，每月结束之后会给活跃度排行前三的人增加积分，用来兑换周边文创。

## 3. 加精（Pin）

- 你的消息被管理员pin后，会计入“被加精次数”。
- 每周一上午 9 点，群里会自动发“上周加精汇总”（仅上周有新增加精时才发送）。


## 4. 单聊机器人

你可以直接单聊机器人：

- 发飞书文档链接（`docx/doc/wiki`）-> 回你摘要图片，当想要分享给其他用户时可以用这个方式。
- 发普通文本 -> 回你文字图片

`[截图建议-05：单聊问答示例]`

## 常见问题（简版）

- 为啥没归档：先看是不是在目标群、有没有带正确 `#标签`。
- 为啥机器人没回：先确认你已添加机器人，再确认你发的是文本或文档链接。
- 为啥统计有延迟：系统是批量更新，短延迟正常。
